# 文字数カウンター

テキストを入力するだけで、文字数をリアルタイムに確認できるWebツールです。

🔗 **公開URL**: https://horis-project.github.io/character-counter/

---

## 機能

- **文字数カウント** — スペース・改行を含む総文字数
- **空白除外文字数** — スペース・改行を除いた実質文字数
- **行数カウント** — 改行で区切られた行の数
- **SNSプリセット** — X / Instagram / note / LINE の文字数上限をワンクリックで設定
- **文字数上限 + プログレスバー** — 上限に対する使用率をリアルタイム表示。超過時は赤色で警告
- **スマートフォン対応** — レスポンシブデザイン

> 入力したテキストはサーバーに送信されません（すべてブラウザ内で処理）

---

## 使い方

1. テキストエリアに文章を入力または貼り付ける
2. 文字数・空白除外文字数・行数がリアルタイムで表示される
3. SNSプリセットボタンまたは手動入力で上限文字数を設定すると、プログレスバーが表示される

## 技術スタック

| 技術 | 用途 |
|------|------|
| HTML5 | ページ構造 |
| CSS3 (Grid / Flexbox) | レイアウト・レスポンシブ対応 |
| Vanilla JavaScript | カウントロジック・UI操作 |

バックエンド・ビルドツール・外部ライブラリは使用していません。
